# dots
A place for my hypr* linux setup dotfiles

The dots are currently provided 'as-is'. Installation scripts and some tidy-up will happen soon (ish).

## Requirements
- Hypr(idle/lock/paper/polkitagent/land)
- Hack Nerd Font
- Wofi
- Pyprland
- Waybar
- Vicinae (optional, swap with waybar for launcher if you don't want it)
- Ghostty
- Swaync

There are many utilities installed that I find useful for my setup, but you may not have/want. These include
- Light (laptop backlight control)
- Solaar (logitech mouse manager)
- Blueman-applet
- Hyprshot (3rd party)
